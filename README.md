1 - Clone the repository

2 - Use node version

```
nvm use
```

3 - Install dependencies

NPM

```
npm install
```

or

PNPM
```
pnpm install
```

4 - Run the boilerplate

NPM

```
npm run dev
```

or

PNPM
```
pnpm run dev
```
